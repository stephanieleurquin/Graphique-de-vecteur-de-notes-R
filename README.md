# Exercice : Visualisation des Notes des Élèves

## Description
Ce projet est un petit exercice en R pour pratiquer :
- La création de vecteurs
- Les calculs statistiques de base (`mean`, `sum`, `max`, `min`)
- La visualisation des données avec un graphique en barres

Le script principal `notes_graph.R` crée un vecteur de notes, calcule les statistiques, puis affiche un **barplot** avec la moyenne en rouge.

---

## Contenu du projet
- `notes_graph.R` : script R contenant le code pour créer le graphique et les statistiques
- `README.md` : ce fichier d'explication

---

## Comment exécuter le script
1. Cloner ou télécharger ce repository
2. Ouvrir RStudio
3. Ouvrir `notes_graph.R`
4. Cliquer sur **Source** ou exécuter le code ligne par ligne
5. Observer les résultats dans la console et le graphique dans le panneau "Plots"

---

## Exemple de sortie
- Moyenne : 13,8  
- Somme : 138  
- Note maximale : 18  
- Note minimale : 9  

Le graphique affiche chaque note avec une barre bleue et la moyenne en ligne rouge.
